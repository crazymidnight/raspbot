Telegram-бот для просмотра расписания занятий Томского Политехнического Университета
------------------------------------------------------------------------------------
### Как пользоваться:
1. Напиши <a href=http://t.me/rasptpubot>боту</a> в Telegram номер своей группы
2. Получи актуальное расписание на сегодня
